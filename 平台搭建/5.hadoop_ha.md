* 启动journalnode
```
hadoop-daemon.sh start journalnode
```
* 格式化active namenode
```
hdfs namenode -format
hadoop-daemon.sh start namenode
```
* 同步standby namenode
```
hdfs namenode -bootstrapStandby
```
* stop active namenode
```
hadoop-daemon.sh stop namenode
```
* 格式化 active namenode zkfc
```
hdfs zkfc -formatZK
```
* 启动hdfs
```
start-dfs.sh
```

* 从非ha集群升级到ha
```
1 复制原namenode中dfs.name.dir下的内容到新增的standby节点相同目录下
2 修改各节点配置
3 active节点执行格式化journal节点hdfs namenode -initializeSharedEdits
4 active节点执行hadoop-daemon.sh start namenode
5 standby 节点执行hdfs namenode -bootstrapStandby
6 active节点执行hadoop-daemon.sh stop namenode
7 active节点执行hdfs zkfc -formatZK
8 active节点执行start-dfs.sh
```
* 注
```
fence失败，可以尝试在各namenode节点安装psmisc(包含fuser)
sudo yum install psmisc
```
